How to Host Your Site for Free using GitHub Pages

Recommendations:

  • Create a GitHub account if you don't have one.
  • Create a new repository on GitHub with the name username.github.io, replacing username with your GitHub username.
  • Upload your website files (HTML, CSS, JavaScript) to the repository.
  • Make sure you have an index.html file in your repository; this will be the default landing page for your site.
  • Go to the repository's Settings, scroll down to the GitHub Pages section, and choose the main branch as the source for your GitHub Pages site.
  • GitHub will provide you with a URL (usually https://username.github.io) where your site will be hosted.
  • Access your site using the provided URL to see your hosted website.
